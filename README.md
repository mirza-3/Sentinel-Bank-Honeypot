# Sentinel-Bank: A Full-Stack Deception & Intrusion Monitoring Framework

## 🎯 Project Overview
Sentinel-Bank is a cybersecurity honeypot system designed to simulate a real-world banking environment to lure, detect, and analyze unauthorized network interactions in real-time.

## 🚀 Key Components
- **The Bait (Front-End):** **ZU-BANK** - A realistic fake banking portal designed to attract potential attackers.
- **The Trap (Deception Engine):** A custom Python-coded backend that monitors socket connections and logs credential harvesting attempts.
- **Honeypot HQ (Dashboard):** A centralized real-time monitor providing visualization of attack data, source IPs, and intercepted credentials.

## 🛠️ Tech Stack & Skills
- **Language:** Python (Socket Programming)
- **Frontend:** HTML5, CSS3
- **Security:** Deception Technology, Intrusion Detection (IDS), Log Analysis
- **AI Integration:** AI-assisted logic for code optimization and threat categorization.

## 📈 Impact
Demonstrates the ability to build a functional end-to-end security monitoring solution that reduces "Time-to-Detect" for reconnaissance attempts.
